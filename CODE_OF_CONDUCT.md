# Code of Conduct

## Purpose

This project aims to foster an open, respectful, and
constructive environment for everyone participating in
the community.

We value technical discussion, collaboration, curiosity,
and mutual respect.

## Expected Behavior

Please:

- Be respectful and professional
- Assume good intent where reasonable
- Accept constructive feedback gracefully
- Keep discussions technical and productive
- Help create a welcoming environment for contributors
  of all experience levels

## Unacceptable Behavior

The following behavior is not acceptable:

- Personal attacks or harassment
- Discrimination or hate speech
- Trolling, baiting, or intentionally disruptive behavior
- Publishing private information without consent
- Repeatedly derailing discussions
- Spam or abusive communication

## Technical Disagreements

Strong technical opinions and disagreements are normal
in engineering projects.

Debate is welcome, but keep it focused on:
- facts
- implementation
- design trade-offs
- measurable outcomes

Avoid making discussions personal.

## Enforcement

Project maintainers may remove, edit, or reject comments,
issues, pull requests, or other contributions that violate
this Code of Conduct.

Repeated or severe violations may result in temporary or
permanent bans from participation.

## Scope

This Code of Conduct applies to:

- Git repositories
- Issues and pull requests
- Discussions and chat platforms
- Community spaces related to this project

## Reporting

If you experience or witness unacceptable behavior,
please contact the project maintainers privately.
